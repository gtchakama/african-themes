# African Themes Collection

A collection of Visual Studio Code themes inspired by the colors and landscapes of Africa.

## Themes

### Rwanda
- **Thousand Hills (Dark)**: Deep blues, forest greens, and sun-drenched yellows, reflecting Rwanda's "Land of a Thousand Hills."
- **Akagera Sunrise (Light)**: A clean, vibrant theme inspired by the morning light over Akagera National Park.

## How to Install

1. Open VS Code.
2. Go to Extensions.
3. Search for "African Themes Collection" (coming soon to Marketplace).
4. Or, clone this repository into your `.vscode/extensions` folder.
