# African Themes Collection

A collection of Visual Studio Code themes inspired by the colors and landscapes of Africa.

## Themes

### Rwanda
- **Thousand Hills (Dark)**: Deep blues, forest greens, and sun-drenched yellows.
- **Akagera Sunrise (Light)**: A clean, vibrant theme inspired by the morning light.

### Kenya
- **Savannah Dusk (Dark)**: Earthy tones and deep reds reflecting the Maasai Mara at sunset.
- **Rift Valley (Light)**: A bright, fresh theme inspired by the Great Rift Valley.

### Ghana
- **Black Star (Dark)**: Bold Pan-African colors (red, gold, green) with deep black accents.
- **Black Star (Light)**: A high-contrast light theme using the Pan-African palette.

### South Africa
- **Rainbow Nation (Dark)**: A vibrant dark theme using the colors of the South African flag.
- **Table Mountain (Light)**: A sophisticated light theme reflecting Cape Town's iconic landscape.

### Zimbabwe
- **Flame Lily (Dark)**: Deep blacks with gold and crimson accents, inspired by the national flower.
- **Great Enclosure (Light)**: A granite-toned light theme inspired by Great Zimbabwe.

### Nigeria
- **Eagle Green (Dark)**: Deep forest greens and whites, reflecting the strength of the Super Eagles.
- **Pearl White (Light)**: A pristine light theme with vibrant green highlights.

### Namibia
- **Desert Moon (Dark)**: Deep blues and golds reflecting the Namib desert under moonlight.
- **Sossusvlei (Light)**: Warm sand tones and deep reds inspired by the iconic dunes.

## How to Install

1. Open VS Code.
2. Go to Extensions.
3. Search for "African Themes Collection".
4. Or, clone this repository into your `.vscode/extensions` folder.
